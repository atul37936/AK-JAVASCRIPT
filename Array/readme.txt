Easy

Print all elements – Create an array of your favorite movies and print each one using a loop.

Sum of array – Given [10, 20, 30, 40], calculate the sum using a loop.

Find the largest – Find the largest number in an array [3, 7, 2, 9, 5].

Count even numbers – Count how many even numbers are in [1,2,3,4,5,6].

Reverse an array – Print the array in reverse order without using .reverse().



Medium

Average of numbers – Calculate average of array elements.

Search for a value – Check if a given value exists in the array (without using .includes()).

Remove duplicates – From [1,2,2,3,4,4,5] create a new array with unique values.

Merge two arrays – Combine two arrays into one.

Second largest – Find the second largest number in an array.




Hard

Sorting (without .sort()) – Implement bubble sort to sort an array.

Palindrome array – Check if an array reads the same forwards and backwards (e.g., [1,2,3,2,1]).

Rotate array – Rotate array left by one position (e.g., [1,2,3,4] → [2,3,4,1]).

Find missing number – Given an array of consecutive numbers missing one, find the missing number.

Matrix addition – Work with 2D arrays (arrays inside arrays).
